Sidebar: The lang attribute

You can set the language with the global lang attribute. This is generally set for the whole page by putting it in the opening <html> tag, but you can set it for any tag.

Some examples are en-US for US English, he for Hebrew, and lb for Luxembourgish. Sadly there is no code for Glagolitic.

lang is not required, but is recommended. When set, screen readers work properly, and the sun shines a little more brightly. It will also make the right decimal separator show on a virtual keyboard when contenteditable="true" and inputmode="decimal". Some countries use a dot, some like their commas.
