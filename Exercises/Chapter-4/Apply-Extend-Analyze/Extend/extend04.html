Summary of CSS color formats

Hex is not human-readable. RGB is a bit better, but still not intuitive. HSL uses hue, saturation, and lightness instead of red, green, and blue components, which is a lot more human-friendly. Unfortunately, it’s not perceptually uniform, and (like RGB and hex) is locked to the sRGB color space. OKLCH uses lightness, chroma (kind of like saturation), and hue; is perceptually uniform, making it easy to work with, and is not locked to any color space. OKLCH is the future. Use OKLCH.

HWB summary: A cylindrical mapping of RGB space, like HSL and HSV. Uses whiteness and blackness instead of saturation and lightness/value.

CMYK summary: A subtractive color model, used for printing. Computer displays use additive color models. RGB, HSL, and OKLCH are all additive.
